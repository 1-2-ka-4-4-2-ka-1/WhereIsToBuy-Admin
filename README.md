# WhereIsToBuy-Admin
Admin panel for WhereIsToBuy(Billing App for stores/shops) Where you can analyse data, manipulate data and can export invoice in desired format.

*Shops Data
*Manage Users
*Manage Items
*Manage Bills
*Filter Bills
*Export as PDF
*Export to Excel
